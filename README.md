# Reverse: 1999 Wallpapers

## What is this?

Detailed classification of official CGs from the game **Reverse: 1999** for user download.

Assets are sourced from [here](https://github.com/myssal/Reverse-1999-CN-Asset).

To visit the HTML page, please click [here](https://reverse-1999-wallpaper.pages.dev/search).

## Notes

The website is still under development. It is currently in **Developer Preview** and may be updated at any time.

The site is deployed on CloudFlare. Due to regional network policies, it may be inaccessible in some areas. Please use a VPN if you encounter connection issues.

## Simple Coding Standards and Guidelines

**Prevent flickering when switching to dark mode**

If you use the `transition` property on an element that has a background color, please add `background-color 250ms` to the `transition` rule.

Detailed code can be found in the `initialize.css` file.
